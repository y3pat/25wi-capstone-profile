# 25wi-capstone-profile

Descriptive website for the **ICU Sepsis Decision Support System** capstone project.

## Structure

- **`index.html`** – Single-page scroll-based layout (GitHub Pages compatible)
- **`css/style.css`** – Styles
- **`js/main.js`** – Smooth scroll, nav highlight, mobile menu
- **`images/`** – Place screenshots and diagrams here

## Sections

Hero → Introduction → Methods → Results → Conclusion → Footer

## Local Preview

Open `index.html` in a browser, or run a simple server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## GitHub Pages

Enable Pages in repo Settings → Pages → Source: main branch, / (root). The site will be at:

`https://<username>.github.io/25wi-capstone-profile/`

## Product Link

Update the product/demo URL in `js/main.js` (`PRODUCT_URL`) and in the `href` attributes of the "Try the App" buttons in `index.html`.
